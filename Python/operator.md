## 연산자 정리

- c언어 내에서 &&, ||, ^, !로 사용하는 연산자를 대체하는 파이썬 문법문이다.

## AND 연산자

```python
if s>10 and s<20:
    ~

if s>10 & s<20:
    ~
```

## OR 연산자

```python
if s>10 or s<20:
    ~

if s>10 | s<20:
    ~
```

## NOT 연산자

```python
if not s==1:
    ~

if s>10 & s<20:
    ~
```

## in, not in 연산자

```python
if not in s<20:
    ~

if s>10 & s<20:
    ~
```
